# Assets

This folder contains images and other multimedia assets used throughout the Temporal Failure-Proof Python course.

## Current Files

- [anchorman_60_percent_API.jpg](anchorman_60_percent_API.jpg) - Anchorman meme image for course content
- [sometimes_we_are_all_jerry.png](sometimes_we_are_all_jerry.png) - meme image acknowledging the authors make bugs
- [works_on_my_machine.png](works_on_my_machine.png) - Temporal "works on my machine" meme for Module 4

## Usage

These assets are referenced in the course assignment files and can be used in markdown files with relative paths:

```markdown
![Alt text](../assets/filename.ext)
```

## File Organization

- Keep image files reasonably sized for web viewing
- Use descriptive filenames
- Supported formats: PNG, JPG, GIF, SVG
- Consider adding new assets to this folder to keep the course materials organized
